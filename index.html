색은 숫자다
RGB 완전 정복

모든 색은 빨강(R), 초록(G), 파랑(B) 숫자 세 개로 만들어져요.
슬라이더를 움직여서 직접 확인해보세요!

1
RGB 슬라이더 — 색은 숫자 3개다
#ff6450 → [255, 100, 80]
R 255
G 100
B 80
R=빨강, G=초록, B=파랑. 각각 0~255 사이 숫자예요.
셋 다 0 → 검정  |  셋 다 255 → 흰색  |  R만 255 → 빨강
2
자주 쓰는 색 팔레트 — 클릭하면 슬라이더에 반영
픽셀 아트에서 색을 너무 많이 쓰면 지저분해 보여요.
5~8가지 정도가 딱 적당해요. 오늘 내가 쓸 색을 골라보세요!
# 파이썬에서 이렇게 써요
선택한_색 = [255, 100, 80]
3

두 색의 RGB 값을 각각 평균 내면 중간 색이 나와요.
(255+77)//2 = 166   (100+150)//2 = 125   (80+255)//2 = 167
4
cmap — 숫자를 색으로 바꾸는 규칙

plt.imshow(하트, cmap='RdPu') 에서 cmap은 colormap의 줄임말이에요.
2D 리스트의 숫자(0~1)를 어떤 색으로 바꿀지 정하는 팔레트예요. 클릭해서 비교해보세요!

plt.imshow(하트, cmap='RdPu')
같은 그림도 cmap만 바꾸면 전혀 다른 느낌이 돼요.
'gray'는 흑백, 'viridis'는 과학 데이터에 많이 쓰고, 'RdPu'·'plasma'는 예쁜 아트에 잘 어울려요.
